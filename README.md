# sti-udinus-backend
Backend aplikasi Sistem Teknik Informatika UDINUS sebagai bagian dari Kerja Praktek. Dibangun oleh Bram Aristyo, project ini menyediakan API untuk manajemen data pengguna, integrasi layanan kampus, serta pengelolaan modul akademik. Teknologi utama: Laravel, MySQL, REST API, dan arsitektur modular.
