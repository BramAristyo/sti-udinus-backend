# **Backend Sistem Teknik Informatika UDINUS**

### *Project Documentation – Bram Aristyo*

Repository ini merupakan dokumentasi dan implementasi backend untuk Sistem Teknik Informatika UDINUS, dikembangkan sebagai bagian dari kegiatan **Kerja Praktek**. Proyek ini bertujuan untuk menyediakan layanan backend yang terstruktur, terstandarisasi, serta mudah diintegrasikan dengan berbagai sistem internal maupun eksternal di lingkungan akademik.
